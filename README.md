# Consultor de Notas
An unofficial grade calculator for students of the Universidad de la Costa (CUC), built to simplify the tracking of academic performance across the four-cut grading system used by the institution.

## What it does

- Calculates accumulated points across 4 grading periods (cortes)
- Projects what score is needed in pending cuts to pass
- Suggests a proportional score distribution across pending cuts
- Provides a performance rhythm indicator
- Remembers your last session locally (no account required)
- Includes quick access to official CUC platforms

## How to use

Open the file directly in any modern browser, or visit the live version at:
https://heyacosta.github.io/consultor-notas-cuc/

No installation, no dependencies, no account required.

## Grading system

Each subject is divided into 4 cuts totaling 5.0 points. The minimum passing score is 3.0.

| Cut | Weight | Max points | Breakdown |
|---|---|---|---|
| Cut 1 | 30% | 1.50 | Activity 10% + Midterm 20% |
| Cut 2 | 30% | 1.50 | Activity 10% + Midterm 20% |
| Cut 3 | 30% | 1.50 | Activity 10% + Midterm 20% |
| Generic Test | 10% | 0.50 | General exam |

## Tech stack

Built with plain HTML, CSS and JavaScript. No frameworks, no dependencies, no build tools required.

The only external resources used are Google Fonts for typography and Google Forms for the feedback inbox.

## Disclaimer

This tool is not official and is not affiliated with the Universidad de la Costa. No data is collected. All calculations happen locally in the user's browser.
